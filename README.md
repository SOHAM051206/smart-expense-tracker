# Smart Expense Tracker

A modern JavaFX-based desktop application for managing personal expenses with a clean and polished user interface.

---

## Overview

Smart Expense Tracker allows users to:

- Track monthly income
- Add and delete expenses
- Categorize expenses (including custom categories)
- Select dates in dd/MM/yyyy format
- Switch between Light and Dark themes
- View total expenses and remaining balance

The application is built using JavaFX and Maven with a focus on clean UI design and usability.

---

## Tech Stack

- Java 17
- JavaFX 21
- Maven
- IntelliJ IDEA
